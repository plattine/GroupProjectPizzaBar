Java Pizza Ordering System (Group Project)

This project is a command-line pizza ordering system developed as part of my Datamatiker education.

The application simulates a pizzeria workflow, including menu handling, order management, and basic customer types.

Features

Command-line interface for placing and managing orders
Menu loaded from CSV file
Order handling and storage
Customer types (normal, VIP, employee)
Basic statistics and order history
File-based persistence (menu, orders, history)
Modular structure using object-oriented principles

Project Structure

app – application entry point
ui – user interface and interaction
model – domain classes (Pizza, Order, Customer, etc.)
service – business logic and file handling
util – helper classes (error handling, sorting, formatting)
